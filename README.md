# Publicacion web - Gestion Metrologica

Esta carpeta contiene una version portable de las apps estaticas:

- dashboard/index.html
- etiquetas/index.html
- etiquetas/Etiquetas_Calibracion_Impresion.pdf

No depende de rutas locales C:\\ ni file:///.

Para acceso restringido a usuarios `@grupodisal.com.ar`, publicar en Azure Static Web Apps con Microsoft Entra ID single-tenant. La configuracion base esta en `staticwebapp.config.json` y los pasos estan documentados en `PUBLICACION_ACCESO_GRUPODISAL.md`.

GitHub Pages sirve para publicacion publica, pero no cumple el requisito de acceso restringido por dominio corporativo.
