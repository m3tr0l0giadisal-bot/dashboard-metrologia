# Identidad visual Grupo Disal

Referencia: intranet corporativa `https://tersuave365.sharepoint.com/sites/IntranetDisal`.

Nota: el sitio requiere autenticacion Microsoft, por lo que el HTML/CSS real no queda disponible sin sesion. El sistema visual se definio a partir de la captura provista de SharePoint y del comportamiento visual de Microsoft 365.

## Tipografia

- Familia principal: `"Segoe UI Variable", "Segoe UI", Arial, sans-serif`.
- Uso: interfaz operativa, filtros, tablas, dashboard, login y etiquetas.
- Titulos: peso 700-750, sin letter-spacing negativo.
- Labels: 12 px, uppercase, peso 800.

## Colores

- Azul corporativo principal: `#253b86`.
- Azul corporativo oscuro: `#1b2e6f`.
- Verde petroleo corporativo/acento: `#006b5c`.
- Verde estado conforme: `#005941`.
- Fondo app: `#f4f6fa`.
- Superficie: `#ffffff`.
- Superficie secundaria: `#eef3f7`.
- Texto principal: `#1f2933`.
- Texto secundario: `#62717b`.
- Borde: `#d8dee8`.

## Estilo

- Inspiracion: SharePoint/Microsoft 365 corporativo.
- Barras superiores compactas, planas y azules.
- Superficies blancas con bordes suaves, radio bajo de 4-6 px.
- Sombras discretas, solo para separar paneles principales.
- Botones primarios en azul corporativo.
- Foco visible en azul para accesibilidad.
- Evitar gradientes decorativos, iconos emoji y paletas fuera de marca.

## Aplicacion

- Dashboard de metrologia: barra superior, KPI, filtros, botones y graficos.
- Login: fondo claro institucional, marca `GRUPO DISAL` visible, sin referencia a sede.
- Etiquetas/QR: encabezados y controles en azul corporativo.
